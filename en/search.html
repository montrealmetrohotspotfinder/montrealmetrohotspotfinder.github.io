Search for a hotspot and find its nearest metro station and related info with this nifty engine below. This engine searches by metro station so you will likely see some results appear more than once; one for every station it is closest to.



Given that Montreal is a predominantly Francophone city, there are a lot of hotspots that contain one or more accented (special) characters. To use accented characters in French, use one of the following combinations below. Apostrophe (') usage is also common in French. To input an apostrophe, type &#39; into the box. Do not use the actual accented symbols or apostrophes in your search when searching for a spot with such characters in its name; the results will not appear that way since HTML has a special way to deal with these characters, and it is through the keyboard combinations below. I encourage you to use them when searching for a spot with a special character or apostrophe in its name, because in some instances, if you do not use them where appropriate, matches will not appear for your given request (example: "Spheretech Saint-Laurent" will not be treated the same way as how it is actually spelled, "Sphèretech Saint-Laurent"). There are some instances where some spots have a well known English name that will appear on this list, and popular abbreviations for some sites will also be accepted when looking for results.

Also, please note that this engine should not serve as a directory for hotspots on the entire Island of Montreal. This engine gets results of the hotspots that are near Montreal Metro stations only, either within walking distance of them or a short bus drive's distance from them, and are also stored on this website in individual pages for each Montreal Metro station.
